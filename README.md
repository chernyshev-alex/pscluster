  Multi serviced oriented application based on akka cluster and integration service bus apache camel.
  Application Layers : 
    Cluster client
    Frontend nodes  
    Backend nodes host services (role based routing)
    Integration layer use akka camel extension

 Prerequisites 
    
   scala 2.11.8
   akka  2.4.4
   apache camel
